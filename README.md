# 💼 ML Portfolio Optimizer

Modern portfolio construction using ML + classical optimization.

## Optimization Methods
1. **Markowitz (Mean-Variance)** - Classic efficient frontier
2. **Risk Parity** - Equal risk contribution
3. **Black-Litterman** - Bayesian views + market equilibrium
4. **ML-Enhanced** - XGBoost return predictions → optimizer
5. **RL-Based (Experimental)** - DQN for dynamic allocation

## Features
- Factor risk decomposition (size, value, momentum, quality)
- Transaction cost optimization
- Rebalancing strategy (monthly vs quarterly)
- Interactive Streamlit web app

## Tech Stack
- Python 3.11, PyPortfolioOpt, cvxpy
- XGBoost, PyTorch (RL)
- Streamlit, Plotly
